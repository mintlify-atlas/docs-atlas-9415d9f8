## What is Paymenter?

Paymenter is an open-source billing platform designed specifically for hosting companies. It automates subscriptions, eliminates billing chaos, and helps grow your hosting business without vendor lock-ins or hidden costs.

Built on Laravel with modern web technologies, Paymenter provides a flexible and robust solution for managing your hosting services, handling payments, and providing exceptional customer support.

## Supported Integrations

### Payment Gateways
- **Stripe** - Credit card and subscription processing
- **PayPal** - PayPal payments and IPN support
- **Mollie** - European payment methods

### Server Platforms
- **Pterodactyl** - Game server management panel
- **cPanel** - Web hosting control panel
- **Plesk** - Web hosting control panel
- **DirectAdmin** - Web hosting control panel
- **Virtualizor** - VPS management panel
- **Virtfusion** - Cloud management platform
- **Convoy** - Server management platform
- **Enhance** - Web hosting control panel

## Requirements

Before installing Paymenter, ensure your server meets these requirements:

- **PHP** 8.3 or higher
- **Composer** for dependency management
- **Web server** Apache or Nginx
- **Database** MariaDB
- **Redis** for caching (recommended)

<Note>
  Paymenter is built on Laravel 12 and follows Laravel's system requirements. Additional PHP extensions like `ext-intl`, `ext-json`, `ext-mbstring`, `ext-pdo`, and `ext-zip` are required.
</Note>
