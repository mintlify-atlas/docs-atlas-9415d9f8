The Orders API allows you to create, retrieve, update, and delete customer orders. Orders group one or more services together that were purchased at the same time.

## The Order Object

<ResponseField name="id" type="integer">
  Unique identifier for the order
</ResponseField>

<ResponseField name="user_id" type="integer">
  ID of the user who placed the order
</ResponseField>

<ResponseField name="currency_code" type="string">
  Three-letter currency code (e.g., USD, EUR)
</ResponseField>

<ResponseField name="created_at" type="datetime">
  Timestamp when the order was created
</ResponseField>

<ResponseField name="updated_at" type="datetime">
  Timestamp when the order was last updated
</ResponseField>

### Relationships

<ResponseField name="user" type="object">
  The customer who placed this order
</ResponseField>

<ResponseField name="services" type="array">
  Services included in this order
</ResponseField>

## List Orders

<CodeGroup>
```bash cURL
curl https://your-domain.com/api/v1/admin/orders \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Accept: application/json"
```

```javascript JavaScript
const response = await fetch('https://your-domain.com/api/v1/admin/orders', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Accept': 'application/json'
  }
});

const orders = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://your-domain.com/api/v1/admin/orders',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Accept': 'application/json'
    }
)

orders = response.json()
```

```php PHP
$response = Http::withHeaders([
    'Authorization' => 'Bearer ' . $apiKey,
    'Accept' => 'application/json',
])->get('https://your-domain.com/api/v1/admin/orders');

$orders = $response->json();
```
</CodeGroup>

### Query Parameters

<ParamField query="per_page" type="integer" default="15">
  Number of orders to return per page (max 100)
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="filter[id]" type="integer">
  Filter orders by ID
</ParamField>

<ParamField query="filter[currency_code]" type="string">
  Filter orders by currency code
</ParamField>

<ParamField query="sort" type="string">
  Sort orders. Available: `id`, `created_at`, `updated_at`, `currency_code`. Prefix with `-` for descending.
</ParamField>

<ParamField query="include" type="string">
  Include related resources: `services`, `user`. Comma-separated.
</ParamField>

## Create an Order

<CodeGroup>
```bash cURL
curl -X POST https://your-domain.com/api/v1/admin/orders \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": 1,
    "currency_code": "USD"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-domain.com/api/v1/admin/orders', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Accept': 'application/json',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    user_id: 1,
    currency_code: 'USD'
  })
});

const order = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://your-domain.com/api/v1/admin/orders',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Accept': 'application/json',
        'Content-Type': 'application/json'
    },
    json={
        'user_id': 1,
        'currency_code': 'USD'
    }
)

order = response.json()
```
</CodeGroup>

### Request Body

<ParamField body="user_id" type="integer" required>
  ID of the user placing the order
</ParamField>

<ParamField body="currency_code" type="string" required>
  Three-letter currency code (e.g., USD, EUR, GBP)
</ParamField>

<Note>
  Creating an order creates an empty order container. You'll need to create services associated with this order separately using the [Services API](/api/services).
</Note>

### Response

```json
{
  "data": {
    "type": "orders",
    "id": "42",
    "attributes": {
      "user_id": 1,
      "currency_code": "USD",
      "created_at": "2024-01-20T15:45:00.000000Z",
      "updated_at": "2024-01-20T15:45:00.000000Z"
    },
    "relationships": {
      "services": {
        "data": []
      }
    }
  }
}
```

## Delete an Order

<CodeGroup>
```bash cURL
curl -X DELETE https://your-domain.com/api/v1/admin/orders/1 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Accept: application/json"
```

```javascript JavaScript
const response = await fetch('https://your-domain.com/api/v1/admin/orders/1', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Accept': 'application/json'
  }
});

// Returns 204 No Content on success
```

```python Python
import requests

response = requests.delete(
    'https://your-domain.com/api/v1/admin/orders/1',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Accept': 'application/json'
    }
)

# Returns 204 No Content on success
```
</CodeGroup>

### Path Parameters

<ParamField path="id" type="integer" required>
  The ID of the order to delete
</ParamField>

### Response

Returns HTTP 204 No Content on success.

<Warning>
  Deleting an order may also affect associated services. Ensure you understand the implications before deleting orders.
</Warning>

## Order Workflow

Typical order creation workflow:

1. **Create Order** - Create an empty order with user and currency
2. **Add Services** - Create services linked to the order
3. **Generate Invoice** - Invoice is automatically created for the services
4. **Process Payment** - Customer pays the invoice
5. **Activate Services** - Services are activated once payment is confirmed

## Finding User Orders

While you can filter by ID, to get all orders for a specific user, use the Users API with includes:

```bash
curl "https://your-domain.com/api/v1/admin/users/1?include=orders" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Recent Orders

Get the most recent orders:

```bash
curl "https://your-domain.com/api/v1/admin/orders?sort=-created_at&per_page=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
